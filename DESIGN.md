# BlockRoom Design Reset

No final visual direction is currently selected.

Before any future design work:

1. Read `docs/project-blueprint.md` in full. It is the source of truth for all
   product behavior and real-state requirements.
2. Read both downloaded references in full:
   - `docs/design-references/cohere/DESIGN.md`
   - `docs/design-references/elevenlabs/DESIGN.md`
3. Wait for the product owner's instruction on whether to use Cohere,
   ElevenLabs, or a deliberate hybrid.

The former BlockRoom visual specification and concept direction are retired.
Current CSS is only the working MVP presentation and may be replaced entirely,
provided every existing function remains intact.
